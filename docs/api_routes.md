# API Routes Documentation

## Authorization

- **POST** `/api/auth/register`
- **POST** `/api/auth/login`
- **POST** `/api/auth/logout`

## Flights

- **GET** `/api/flights`
- **GET** `/api/flights/:id`
- **POST** `/api/bookings/flight`

## Hotels

- **GET** `/api/hotels`
- **GET** `/api/hotels/:id`
- **POST** `/api/bookings/hotel`

## Bookings

- **GET** `/api/bookings`
- **DELETE** `/api/bookings/:id`
